Ibis

A lightweight template engine in Python. Useful as a drop-in component in text-processing applications.

Sample syntax:

<ul>
    {% for post in posts %}
        <li><a href="{{ post.url }}">{{ post.title }}</a></li>
    {% endfor %}
</ul>

Features:

  • Django/Jinja-style syntax.
  • Supports looping, conditionals, filters, template inheritance.
  • Extensible. Easily add custom template tags and filters.
  • Self-contained, no dependencies. Use as a drop-in component in any project.
  • Public domain code. No license compatibility issues.

Installation

You can install Ibis directly from the Python Package Index using pip:

$ pip install ibis

Alternatively, you can download the latest version from the project's Github homepage and run:

$ python setup.py install

Ibis requires Python 3.2 or later.
